Hi!
My name is Luis U. Kentzler.
I’m a UI + UX visual designer.

I like to define myself as a UI + UX designer because I think the UI/UX term blurs the line between UI and UX. Even though they are very different fields, they converge on the creation of a great user interface that maximizes user experience. I use visual design to make great UI with excellent UX.

You can learn more about my process here.

My Motto

    2. My goal is to communicate your content in the right way to your audience. Any User Interface needs the content before starting the design. You’ll never see me design with Lorem Ipsum.
